MICROSERVICIO MS-DONACIONES
────────────────────────────
URL: https://github.com/oscar-flores-leroy/ms-donaciones
Puerto: 8081
Patrones: Repository Pattern + Factory Method (GoF)
Persistencia: Spring Data JPA + H2 (archivo) — ./data/donaciones-db
Documentación API: Swagger UI en /swagger-ui.html
Descripción: Gestión del ciclo de vida completo de donaciones
             (ROPA, ALIMENTO, MEDICO, HIGIENE). Incluye pruebas
             unitarias con JUnit 5 + Mockito (Service, Controller,
             Factories, Repository JPA) y reporte JaCoCo.
Ramas:
  - main
  - develop
  - feature/ms-donaciones-repository
  - feature/ms-donaciones-factory
  - feature/ms-donaciones-jpa-persistence
  - feature/ms-donaciones-tests
